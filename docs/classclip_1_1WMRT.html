Detailed Description

Weighted MRT conversion and utility definitions for Lattice Boltzmann Method.

The WMRT class contains conversion matrices for D2Q9 and D3Q19 lattices, mapping distribution functions to weighted moment space and back. It also stores velocity set definitions and structured maps for applying different types of boundary conditions.
